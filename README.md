# Quantum-Inspired Compression of Large Language Models for Edge Intelligence

This repository contains evaluation code used in the paper:

"Quantum-Inspired Compression and Distillation of Large Language Models for Edge Intelligence"

## Description
The code evaluates classification performance and inference latency
of a quantum-inspired compressed language model under simulated edge constraints.

## Files
- evaluation_results.py : Accuracy, confusion matrix, and latency evaluation

## Requirements
- Python 3.8+
- NumPy
- Matplotlib
- scikit-learn

## Usage
```bash
python evaluation_results.py
